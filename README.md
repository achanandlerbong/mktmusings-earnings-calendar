# Market Musings — Earnings Calendar

Public iCal feed of next earnings dates for the Market Musings coverage universe.

## Subscribe

**Raw URL:** https://raw.githubusercontent.com/achanandlerbong/mktmusings-earnings-calendar/main/earnings_calendar.ics

**Calendar.app:** File → New Calendar Subscription → paste the URL above → set auto-refresh to "Every day".

**Google Calendar:** Other calendars → From URL → paste the URL.

Updated automatically by `pipelines/earnings-calendar/sync.py` in the source repo.
